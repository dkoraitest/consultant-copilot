# Consultant Copilot

AI-копилот для бизнес-консультанта с поэтапным развёртыванием.

## Что это?

Система для автоматизации работы консультанта:
- **Суммаризация встреч** — автоматическое создание протоколов по типу встречи
- **RAG Copilot** — ответы на вопросы по базе знаний (1100+ встреч)
- **Интеграции** — Todoist, Fireflies, Telegram

## Этапы развёртывания

| Этап | Описание | Статус |
|------|----------|--------|
| 1 | Суммаризация встреч (замена Make) | В разработке |
| 2 | RAG Copilot (Q&A по базе знаний) | Планируется |
| 3 | Graph RAG + аналитика | Планируется |

## Быстрый старт

### 1. Клонировать и настроить

```bash
cd Consalt-Copilot
cp .env.example .env
# Заполнить переменные в .env
```

### 2. Запустить через Docker

```bash
docker-compose up -d
```

### 3. Настроить webhook

Указать URL webhook в Fireflies:
```
https://your-server.com/api/webhook/fireflies
```

## Типы встреч

| Тип | Назначение | Лимит | Особенности |
|-----|------------|-------|-------------|
| Working | Рабочая встреча с командой | 3600 символов | Имена разрешены |
| Diagnostics | Диагностика с новым клиентом | - | 15 вопросов, обращение на "вы" |
| Traction | Еженедельный трекшн | 4000 символов | Без имён, фокус на гипотезы |
| Intro | Первое знакомство | - | Короткий формат |

## Технический стек

- **Python 3.11+**
- **PostgreSQL 15** + pgvector
- **FastAPI** — API
- **python-telegram-bot** — Telegram бот
- **Claude API** — LLM для суммаризации
- **LangChain** — RAG (этап 2)
- **Docker Compose** — деплой
